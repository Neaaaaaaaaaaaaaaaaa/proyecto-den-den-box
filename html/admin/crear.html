Crear usuario

Registrar o actualizar un usuario del sistema y asignarlo a un inmueble ya creado.

Este formulario no crea inmuebles nuevos. Solo asigna el usuario al inmueble que ya exista en la base de datos.

Si eliges "Sí", el sistema lo registrará también como residente del mismo inmueble.

Personas en el inmueble

Registra hasta 3 adultos y 3 menores asociados al apartamento.
